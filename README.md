# CS-480-Final

## Libraries:

- GLEW
- GLFW
- GLM
- SOIL2
- Assimp

## CONTROLS:

### Planetary observation mode:

- W - move forward
- S - move backward
- A - move left
- D - move right
- MOUSE - move camera
- LSHIFT - toggle planet panning mode
- SPACE - switch to exploration mode

### Exploration mode:

- W - accelerate forward
- S - accelerate backward
- A - tilt left
- D - tilt right
- Q - tilt down
- E - tilt up
- Z - roll left
- C - roll right
- LSHIFT - brake
- SPACE - switch to planetary observation mode
